# Projeto iFood - Automação de Análise de Notícias

Sistema automatizado para análise de notícias sobre o iFood e concorrentes, incluindo:
- Identificação de porta-vozes
- Análise de protagonismo usando LLM
- Detecção de notas oficiais
- Identificação de estabelecimentos que usam delivery
- Consolidação e geração de relatórios

## Requisitos

- Python 3.8+
- Conta Google com acesso ao Google Drive
- API Key do DeepSeek
