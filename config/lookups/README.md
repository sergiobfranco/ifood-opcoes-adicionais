# Arquivos de Lookup

Esta pasta contém arquivos de referência para lookups de IDs.

## Estrutura

- `nivel_protagonismo/`: Coloque aqui o arquivo com IDs de níveis de protagonismo
- `esforco/`: Coloque aqui o arquivo com IDs de tipos de esforço  
- `nota/`: Coloque aqui o arquivo com IDs de tipos de nota

## Regras

- Apenas **um arquivo .xlsx** por pasta
- O nome do arquivo não importa
- O sistema usará automaticamente o primeiro `.xlsx` encontrado
